# Calorie-Counter

A lightweight JavaScript application to track daily food intake and calories.

## Features
- Add food items with calorie counts.
- Real-time total calorie calculation.
- Delete individual entries.
- Persistent storage using `localStorage`.
- Responsive UI built with Tailwind CSS.
- Simulated API integration via Fetch.

## Setup Instructions
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. No build step required (Tailwind CSS is loaded via CDN).

## Project Structure
- `index.html`: UI Structure.
- `script.js`: Core logic and DOM manipulation.
- `localStorage`: Used to persist data across refreshes.

## How to Run
1. Clone the repository
2. Open index.html in your browser
